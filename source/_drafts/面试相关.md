## Linux 命令
> 系统相关
```bash
# 查看系统相关信息
uname -a
Linux devil 5.8.0-48-generic #54~20.04.1-Ubuntu SMP Sat Mar 20 13:40:25 UTC 2021 x86_64 x86_64 x86_64 GNU/Linux
# 系统名 主机名 操作系统版本 内核版本 ... 操作系统
```

```bash
# 查看系统运行状况
uptime
 14:27:29 up  1:30,  1 user,  load average: 0.39, 0.44, 0.54
# 当前时间 主机运行时长 在线用户数 平均负载 1min 5min 15min
# 平均负载是指单位时间内，系统处于可运行状态和不可中断状态的平均进程数，也就是平均活跃进程数，它和 CPU 使用率并没有直接关系
```
