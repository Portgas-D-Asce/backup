## 开启Vim编程之旅
### 使用 vim 编写程序
**模式：**
- 普通模式：按 esc 从 插入模式 退出到普通模式；
- 插入模式：按 i 进入；
- 命令模式：打一个 : 即可进入命令模式：
  - wq：保存并退出（快捷键：ZZ）； 
  - dd：删除一行；
  - u：撤销操作；

**linux 终端 tab键 自动补全**
```
//输入
gcc main tab键
//补全为
gcc main.c
```

**linux cat 命令**
- 作用：将文件内容打印到标准输出设备上；
```
cat main.c
//常用参数
-n: 同时输出行号；
-b: 同 n，但是空白行不编号； 
```
