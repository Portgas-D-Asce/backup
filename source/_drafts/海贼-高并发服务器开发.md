# 分布式服务器监控系统

## MySQL
### 安装
```bash
sudo apt install mysql-server
```
### 判断 mysql 是否启动
```bash
ps -ef | grep mysql

# 查看服务器状态
systemctl status mysql.service
```
### 初始化配置
```bash
sudo mysql_secure_installation
```

### 登录/退出
```bash
sudo mysql -u root -p

exit
```

### 创建用户/授权/删除用户
```bash
# 创建用户
create user 'user'@'host(%)' identified by 'password';

# 赋予权限, 被授权用户不能向其它人授权
grant privileges on database.table to 'user'@'host'；

# 可以向其它人授权
grant privileges on database.table to 'user'@'host' with grant option；

# 撤销权限
revoke privileges on database.table from 'user'@'host';

# 删除用户
drop user 'user'@'host';
```

### 启动/关闭/重启
```bash
sudo service mysql start

sudo service mysql stop

sudo service mysql restart
```

## 数据库可视化工具
> 连接不上问题
解决方案: 在文件 /etc/mysql/mysql.conf.d/mysqld.cnf 中隐藏以下内容:
```bash
# bind-address          = 127.0.0.1
# mysqlx-bind-address   = 127.0.0.1
```
## C语言连接 mysql
> 安装开发包
```bash
sudo apt install libmysqlclient-dev
```
遇到 zlib1g-dev 问题：[解决方案](https://www.cnblogs.com/aaron-agu/p/8862624.html)

> 测试代码
```cpp
#include <stdio.h>
#include <mysql.h>

int main() {
    MYSQL *mysql = NULL;
    if((mysql = mysql_init(NULL)) == NULL) {
        fprintf(stderr, "error int mysql_init!\n");
        exit(1);
    }
    mysql = mysql_real_connect(mysql, "101.200.202.190", "pk", "pk1024", "mysql", 0, 0, 0);
    if(!mysql) {
        fprintf(stderr, "connection failed!\n");
        exit(1);
    }
    printf("success!\n");
    return 0;
}
```
> 编译命令
```bash
gcc -I /usr/include/mysql/ mysql.c -lmysqlclient
```
